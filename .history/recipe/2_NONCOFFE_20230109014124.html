HOT 레시피

메뉴명: HOT)초코라떼

메뉴명: HOT)군고구마라떼

ICE 레시피

메뉴명: ICE)딸기라떼

1. 컵에 딸기리플잼 1큰술 + 딸기 슬라이스 4큰술 넣는다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 우유을 85% 붓는다 ; 4. 뚜껑을 닫고 재료들을 섞어준다.

메뉴명: ICE)로얄밀크버블티

1. 펄을 전자레인지에 넣고 40초간 해동시킨다. 2. 컵에 해동한 펄 + 얼음 80%를 채운다. ; 3. 밀크티베이스 80% 붓는다. ; (단, 덜달게는 흰우유 100ml와 섞는다.)

메뉴명: ICE)타로버블티

1. 펄을 전자레인지에 넣고 40초간 해동시킨다. 2. 컵에 해동한 펄 + 얼음 80%를 채운다. ; 3. 계량컵에 타로파우더 4작은술 + 뜨거운물을 섞는다. 4. 컵에 우유 70% + 뜨거운 물에 녹인 타로파우더를 넣는다. ; 5. 뚜껑을 닫고 컵에 넣은 재료들을 섞는다.
