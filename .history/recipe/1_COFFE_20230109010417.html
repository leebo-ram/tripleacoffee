HOT 레시피

메뉴명: HOT)아메리카노(오리지널)

1. 오리지널 원두로 2샷을 내린다. ; 2. 컵에 따뜻한 물을 80% 채운다. ; 3. 2샷을 붓는다.

메뉴명: HOT)아메리카노(스페셜)

1. 스페셜 원두로 2샷을 내린다. ; 2. 컵에 따뜻한 물을 80% 채운다. ; 3. 2샷을 붓는다.

메뉴명: HOT)아메리카노(디카페인)

1. 디카페인 원두로 2샷을 내린다. ; 2. 컵에 따뜻한 물을 80% 채운다. ; 3. 2샷을 붓는다.

메뉴명: HOT)헤이즐넛아메리카노

1. 2샷을 내린다. ; 2. 컵에 헤이즐넛시럽 3펌프(30g)을 넣는다. ; 3. 따뜻한 물을 80% 채운다. ; 3. 2샷을 붓는다.

메뉴명: HOT)카페라떼

1. 2샷을 내린다. ; 2. 우유 300ml를 스팀한다. ; 3. 컵에 2샷을 넣는다. ; 4. 스팀한 우유를 컵에 붓는다.

메뉴명: HOT)바닐라라떼

1. 2샷을 내린다. ; 2. 우유 300ml를 스팀한다. ; 3.컵에 바닐라시럽 3펌프(30g)을 넣는다 ; 4. 컵에 2샷을 넣는다. ; 5. 스팀한 우유를 컵에 붓는다.

메뉴명: HOT)헤이즐넛라떼

1. 2샷을 내린다. ; 2. 우유 300ml를 스팀한다. ; 3.컵에 헤이즐넛시럽 3펌프(30g)을 넣는다 ; 4. 컵에 2샷을 넣는다. ; 5. 스팀한 우유를 컵에 붓는다.

메뉴명: HOT)카페모카

1. 2샷을 내린다. ; 2. 초코베이스 360ml를 스팀한다. ; 3. 컵에 2샷을 넣는다. ; 4. 스팀한 초코베이스를 컵에 붓는다.

메뉴명: HOT)카라멜마끼아또

1. 2샷을 내린다. ; 2. 우유 240ml를 스팀한다. ; 3.컵에 카라멜소스 1.5펌프(30g) + 카라멜시럽 2펌프(20g) + 2샷을 넣는다. ; 4. 스팀한 우유를 컵에 붓는다. ; 5. 거품을 2cm 넣는다. 6. 거품 위에 카라멜소스 드리즐를 한다.

메뉴명: HOT)연유커피

1. 2샷을 내린다. ; 2. 우유 300ml를 스팀한다. ; 3.컵에 연유 60g + 2샷을 넣고 섞는다. ; 4. 스팀한 우유를 컵에 붓는다. (단, 덜달게는 연유를 50g 넣는다.)

메뉴명: HOT)민트모카

1. 2샷을 내린다. ; 2. 초코베이스 360ml를 스팀한다. ; 3. 컵에 민트시럽을 2펌프(20g) 넣는다. ; 4. 스팀한 초코베이스 + 2샷을 컵에 붓는다.

ICE 레시피

메뉴명: ICE)아메리카노(오리지널)

1. 오리지널 원두로 2샷을 내린다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 물을 85% 붓는다. ; 4. 2샷을 붓는다.

메뉴명: ICE)아메리카노(스페셜)

1. 스페셜 원두로 2샷을 내린다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 물을 85% 붓는다. ; 4. 2샷을 붓는다.

메뉴명: ICE)아메리카노(디카페인)

1. 디카페인 원두로 2샷을 내린다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 물을 85% 붓는다. ; 4. 2샷을 붓는다.

메뉴명: ICE)헤이즐넛아메리카노

1. 2샷을 내린다. ; 2. 컵에 헤이즐넛시럽 4펌프(40g)을 넣는다. ; 3. 컵에 얼음을 80% 채운다 ; 4. 컵에 물을 85% 채운다. ; 5. 2샷을 붓는다.

메뉴명: ICE)카페라떼

1. 디카페인 원두로 2샷을 내린다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 물을 85% 붓는다. ; 4. 2샷을 붓는다.

메뉴명: ICE)바닐라라떼

1. 2샷을 내린다. ; 2.컵에 바닐라시럽 4펌프(40g)을 넣는다. 3. 컵에 우유을 85% 채운다. ; 3. 2샷을 넣는다. ; 4. 뚜껑을 닫고 컵에 넣을 재료들을 섞는다.

메뉴명: ICE)헤이즐넛라떼

1. 2샷을 내린다. ; 2.컵에 헤이즐넛시럽 4펌프(40g)을 넣는다. 3. 컵에 우유을 85% 채운다. ; 3. 2샷을 넣는다. ; 4. 뚜껑을 닫고 컵에 넣을 재료들을 섞는다.

메뉴명: ICE)카페모카

1. 2샷을 내린다. ; 2. 컵에 얼음을 80% 채운다 ; 3. 컵에 초코베이스를 85% 채운다. ; 3. 2샷을 붓는다.

메뉴명: ICE)카라멜마끼아또

1. 2샷을 내린다. ; 2. 컵에 얼음을 70% 넣는다. ; 3. 우유거품기에 우유를 붓고 거품을 충분히 내어준 후 컵에 붓는다. ; 4.스테인레스 계량컵에 카라멜소스 2펌프(40g) + 카라멜시럽 2펌프(20g) + 2샷을 넣는다 ; 5. 스팀한 우유를 컵에 붓는다. ; 6. 7. 거품 위에 카라멜소스 드리즐를 한다.

메뉴명: ICE)연유커피

1. 2샷을 내린다. ; 2. 컵에 얼음을 80% 넣는다. ; 3.얼음컵에 우유를 70% 넣는다. ; 4.스테인레스 계량컵에 연유 70g + 2샷을 넣고 연유를 녹인다. ; 5. 컵에 담은 후 뚜껑을 닫고 섞는다. (단, 덜달게는 연유를 60g 넣는다.)

메뉴명: ICE)아인슈페너

1. 2샷을 내린다. ; 2. 컵에 얼음을 80% 넣는다. ; 3.얼음컵에 우유를 60%를 붓는다. 4. 크림베이스를 80% 붓는다. 4. 2샷을 넣는다.

메뉴명: HOT)민트모카

1. 2샷을 내린다. ; 2. 컵에 민트시럽 2펌프(20g) + 얼음을 80% 넣는다. ; 3. 얼음컵에 초코베이스를 80% 붓는다. ; 4. 2샷을 넣는다.
