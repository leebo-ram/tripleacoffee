레시피1

메뉴명: HOT)아메리카노(오리지널)

1. 컵에 뜨거운물 85%; + 2. 오리지널 원두 2샷;

메뉴명: HOT)아메리카노(디카페인)

1. 컵에 뜨거운물 85%; + 2. 디카페인 원두 2샷;

메뉴명: HOT)카페라떼

1. 컵에 원두 2샷; + 2. 스팀우유 280mL;

메뉴명: HOT)헤이즐넛라떼

1. 컵에 헤이즐넛시럽 3펌프(30g) (단, 덜달게는 시럽 2펌프(20g));+ 2.원두 2샷; + 3. 스팀우유 280mL;

메뉴명: HOT)카라멜마끼아또

1. 컵에 카라멜소스 1.5펌프(30g) + 카라멜시럽 2펌프(20g) (단, 덜달게는 시럽만 1펌프(10g)) + 원두 2샷 ☞ 풀어주기; + 2. 컵에 스팀우유 거품 90%; + 3. 카라멜소스 드리즐;

메뉴명: ICE)아인슈페너

1. 컵에 얼음 80% + 찬물 50% + 원두 2샷; + 2. 크림베이스 100%;

레시피2

메뉴명: ICE)아메리카노(오리지널)

1. 컵에 얼음 80% + 2.찬물 80%; + 3. 오리지널 원두 2샷;

메뉴명: ICE)아메리카노(디카페인)

1. 컵에 얼음 80% + 2.찬물 80%; + 3. 디카페인 원두 2샷;

메뉴명: ICE)카페라떼

1. 컵에 얼음 80% + 2.우유 80%; + 3. 원두 2샷;

메뉴명: ICE)헤이즐넛라떼

1. 컵에 바닐라시럽 4펌프(40g) (단, 덜달게는 시럽 2펌프(20g));+ 2. 우유 280mL; + 3.원두 2샷; + 4.섞는다.

메뉴명: ICE)카라멜마끼아또

1. 스텐컵에 카라멜소스 2펌프(40g) + 카라멜시럽 2펌프(20g) (단, 덜달게는 시럽만 1펌프(10g)) + 원두 2샷 ☞ 풀어주기; + 2. 컵에 얼음 70% + 우유 거품 90%; + 3. 카라멜소스 드리즐;

메뉴명: HOT)민트모카

1. 컵에 원두 2샷 + 민트초코 파우더 4작은술 + 초코소스 1펌프(20g) (단, 덜달게는 초코소스만 0.5펌프(10g)) + 그린민트시럽 2펌프(40g) ☞ 풀어주기; + 2. 스팀우유 280mL;

레시피3

메뉴명: HOT)아메리카노(스페셜)

1. 컵에 뜨거운물 80%; + 2. 스페셜 원두 2샷;

메뉴명: HOT)헤이즐넛아메리카노

1. 컵에 헤이즐넛시럽 3펌프(30g) (단, 덜달게는 시럽 2펌프(20g)); + 2. 뜨거운물 80%; + 3. 원두 2샷;

메뉴명: HOT)바닐라라떼

메뉴명: HOT)카페모카

메뉴명: HOT)연유커피

1. 컵에 연유 60g(단, 덜달게는 연유 50g) + 원두 2샷 ☞ 풀어주기; + 2. 스팀 우유 280mL

메뉴명: ICE)민트모카

1. 컵에 얼음 80% + 우유 80%; 2. 스텐컵에 민트초코 파우더 4작은술 + 초코소스 1펌프(20g) (단, 덜달게는 초코소스 0.5펌프(10g)) + 그린민트 시럽 1펌프(10g) + 원두 2샷 ☞ 풀어주기;
