마침

레시피1

메뉴명: ICE)딸기라떼

1. 컵에 딸기리플잼 1큰술 + 딸기슬라이스 4큰술 + 딸기 시럽 1펌프(10g); + 2. 컵에 얼음 80% + 우유 100%; + 3.그라데이션을 주면서 섞는다;

메뉴명: ICE)흑당버블티

1. 타피오카펄 1스쿱 👉 전자레인지 40초; + 2. 컵에 흑당소스 65g + 우유 100% 👉 풀어주기;

메뉴명: ICE)군고구마라떼

1. 컵에 얼음 80% + 군고구마베이스 100% (단, 덜달게는 우유 10% + 군고구마베이스 90%);

메뉴명: ICE)로얄밀크티

1. 컵에 얼음 80% + 밀크티베이스 100% (단, 덜달게는 우유 10% + 밀크티베이스 90%);

메뉴명: ICE)쑥라떼

1. 스텐컵에 쑥 파우더 4작은술 + 설탕시럼 2펌프(20g) (단, 덜달게는 설탕시럽만 1펌프(10g)) + 뜨거운물 👉 풀어주기; + 2. 컵에 얼음 80% + 우유 80%

레시피2

메뉴명: ICE)로얄밀크버블티

1. 타피오카펄 1스쿱 👉 전자레인지 40초; + 2. 컵에 얼음 80%; + 3. 밀크티 베이스 100%;

메뉴명: HOT)초코라떼

1. 컵에 스팀 초코베이스 360mL (단, 덜달게는 우유 100mL + 초코베이스 260mL / 휘핑추가시 스팀 초코베이스는 300mL + 초코소스 드리즐);

메뉴명: HOT)녹차라떼

1. 컵에 스팀 녹차베이스 360mL (단, 덜달게는 우유 100mL + 녹차베이스 260mL);

메뉴명: HOT)오곡라떼

1. 컵에 오곡파우더 5작은술 + 설탕시럽 2펌프(20g) (단, 덜달게는 설탕시럽 1펌프(10g)) + 뜨거운물 👉 풀어주기; + 2. 스팀우유 280mL

레시피3

메뉴명: ICE)타로버블티

1. 타피오카펄 1스쿱 👉 전자레인지 40초; + 2. 컵에 얼음 80% + 우유 80%; + 3. 스텐컵에 타로파우더 4작은술 + 뜨거운물 👉 풀어주기;

메뉴명: ICE)초코라떼

1. 컵에 얼음 80% + 초코베이스 100% (단, 덜달게는 우유 10% + 초코베이스 90% / 휘핑추가시 초코베이스는 80% + 초코소스 드리즐);

메뉴명: ICE)녹차라떼

1. 컵에 얼음 80% + 녹차베이스 100% (단, 덜달게는 우유 10% + 녹차베이스 90%);

메뉴명: ICE)오곡라떼

1. 컵에 얼음 80% + 우유 85% + 2. 스텐컵에 오곡파우더 4작은술 + 설탕시럽 1.5펌프(15g) (단, 덜달게는 설탕시럽 1펌프(10g)) + 뜨거운물 👉 풀어주기;

레시피4

메뉴명: ICE)아메리카노(스페셜)

1. 컵에 얼음 80%; + 2.찬물 85%; + 3. 스페셜 원두 2샷;

메뉴명: HOT)군고구마라떼

1. 컵에 스팀 군고구마베이스 360mL (단, 덜달게는 우유 100mL + 군고구마베이스 260mL)

메뉴명: HOT)로얄밀크티

1. 컵에 스팀 밀크티베이스 360mL (단, 덜달게는 우유 100mL + 밀크티베이스 260mL);

메뉴명: HOT)쑥라떼

1. 컵에 쑥 파우더 4작은술 + 설탕시럼 2펌프(20g) (단, 덜달게는 설탕시럽만 1펌프(10g)) + 뜨거운물 👉 풀어주기; 2. 스팀우유 300mL
