HOT 레시피

메뉴명: HOT)아메리카노(오리지널)

1.컵에 따뜻한 물 80% + 2.오리지널 원두 2샷

메뉴명: HOT)아메리카노(스페셜)

1.컵에 따뜻한 물 80% + 2.스페셜 원두 2샷

메뉴명: HOT)아메리카노(디카페인)

1.컵에 따뜻한 물 80% + 2.디카페인 원두 2샷

메뉴명: HOT)헤이즐넛아메리카노

컵에 2샷 + 헤이즐넛시럽 3펌프(30g) + 따뜻한 물을 80%

메뉴명: HOT)카페라떼

컵에 2샷 + 스팀우유 300mL ;

메뉴명: HOT)바닐라라떼

컵에 2샷 + 바닐라시럽 3펌프(30g) + 스팀우유 300mL

메뉴명: HOT)헤이즐넛라떼

컵에 2샷 + 바닐라시럽 3펌프(30g) + 스팀우유 300mL

메뉴명: HOT)카페모카

컵에 2샷 + 스팀 초코베이스 360mL (단, 덜달게는 흰우유 100mL와 섞는다.)

메뉴명: HOT)카라멜마끼아또

컵에 카라멜소스 1.5펌프(30g) + 카라멜시럽 2펌프(20g) + 2샷 + 스팀우유 240mL + 거품 2cm + 거품 카라멜소스 드리즐

메뉴명: HOT)연유커피

컵에 2샷 + 연유 60g + 스팀우유 300mL (단, 덜달게는 연유 50g.)

메뉴명: HOT)민트모카

컵에 2샷 + 그린민트시럽 2펌프(20g) + 스팀 초코베이스 360mL

ICE 레시피

메뉴명: ICE)아메리카노(오리지널)

컵에 얼음을 80% + 물을 85% + 오리지널 원두 2샷

메뉴명: ICE)아메리카노(스페셜)

컵에 얼음을 80% + 물을 85% + 스페셜 원두 2샷

메뉴명: ICE)아메리카노(디카페인)

컵에 얼음을 80% + 물을 85% + 디카페인 원두 2샷

메뉴명: ICE)헤이즐넛아메리카노

컵에 헤이즐넛시럽 4펌프(40g) + 얼음 80% + 물 85% +2샷

메뉴명: ICE)카페라떼

컵에 얼음 80% + 우유을 85% + 2샷

메뉴명: ICE)바닐라라떼

컵에 바닐라시럽 4펌프(40g) + 얼음 80% + 우유 90% + 2샷 + 뚜껑 닫고 재료 섞기

메뉴명: ICE)헤이즐넛라떼

컵에 헤이즐넛시럽 4펌프(40g) + 얼음 80% + 우유 90% + 2샷 + 뚜껑 닫고 재료 섞기

메뉴명: ICE)카페모카

컵에 얼음 80% + 초코베이스 85% + 2샷

메뉴명: ICE)카라멜마끼아또

컵에 얼음 70% + 우유거품 + 스텐컵에 카라멜소스 2펌프(40g) + 카라멜시럽 2펌프(20g) + 2샷 + 카라멜소스 드리즐 ;

메뉴명: ICE)연유커피

컵에 얼음 80% + 우유를 70% + 스텐컵에 연유 70g + 2샷 + 컵에 담은 후 뚜껑을 닫고 섞는다. (단, 덜달게는 연유를 60g 넣는다.)

메뉴명: ICE)아인슈페너

컵에 2샷 + 얼음 80% + 물 60% + 크림베이스 85%

메뉴명: HOT)민트모카

컵에 그린민트시럽 2펌프(20g) + 얼음 80% + 초코베이스 80% + 2샷
